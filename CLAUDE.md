# Контекст проекта

## О проекте
Многомодульная распределённая система. Модули:
- Многопользовательский блокчейн (текущий фокус)
- Синхронизация блокчейнов
- Отображение текста в 3D и связи между текстами
- Экономика в блокчейне
- Универсальная распределённая база знаний

## Текущий фокус
Модуль блокчейна. Спецификация: `docs/blockchain.md`.
Этот документ — источник истины. При расхождениях между кодом и
документом — корректируется код, либо обновляется документ через явное обсуждение.

## Технологический стек
- Язык: C++17
- Сборка: CMake
- Среда разработки: Docker (Ubuntu LTS)
- Криптография: libsodium
- Хранилище: LMDB
- Сериализация: CBOR с deterministic encoding (RFC 8949 §4.2.1)
- Тесты: GoogleTest

## Конвенции кода
- Стандарт C++17, без зависимости от C++20+ возможностей.
- Стиль: snake_case для функций и переменных, PascalCase для классов и типов.
- Явное `noexcept` где применимо.
- Никаких голых `new`/`delete` — только умные указатели и RAII.
- Все криптоопераци через обёртки в `crypto::` namespace, не напрямую libsodium.
- Тесты обязательны для каждой публичной функции модуля.

## Стиль работы
- Перед написанием нового кода — обсудить интерфейс.
- Перед существенными решениями (структуры данных, протоколы) — обновить
  соответствующий документ в `docs/`.
- Открытые вопросы из документов — не "решать молча", а явно обсуждать.
- Документы обновляются вместе с кодом, не отдельно.
